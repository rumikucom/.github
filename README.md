# PT Rumi Kultura Utopia

Welcome to the official GitHub organization of RUMIKU. We build digital platforms, automated tools, and scalable ecosystems that bridge modern technology with global fandom culture.

---

## Our Digital Ecosystem

* **Creedigo**: An all in one digital top up platform providing seamless gaming and utility services.
* **Glocult**: A fandom media portal powered by robust backend scraping and automated content workflows.
* **ROKU Project**: A dedicated retail project specializing in creative fan merchandise and custom goods.
* **Kiyoumi**: An official merchandise store focusing strictly on licensed products and premium brand partnerships.
